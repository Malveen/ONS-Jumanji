Once you have pulled the latest repository and checked out into this branch.
Commands to run to get flask site working.

SET UP VIRTUAL ENVIRONMENT (Not required but recommended):
python3 -m venv flask  "Creates virtual environment"
source flask/bin/activate   "Starts virtual environment"

pip install -r requirements.txt

python3 run.py
